CSV Format

CSV files should include these columns:

  • timestamp, log_type, level, client_ip
  • user_id, method, endpoint, status_code
  • response_time_ms, user_agent, message
  • sql_query, query_duration_ms, server, tenant_id
JSON Format

JSON files should be structured as:

  • One JSON object per line (JSONL)
  • Include timestamp field
  • Include log_type, level, message fields
  • Additional fields are optional
